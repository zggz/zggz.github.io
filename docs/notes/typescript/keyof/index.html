    Keyof
    • keyof
        • 获取类型
        • 索引签名
      • 常用场景

    keyof #

    该操作符用于获取某种类型的所有键,其返回的类型是联合类型

    获取类型 #

    1interface Ponit{
2    x:number
3    y:number
4}
5
6type P = keyof Ponit // 'x' | 'y'

    索引签名 #

    1type Arrayish = {
2    [n:number]: unknown
3}
4type A = keyof Arrayish // number

    常用场景 #

    定义一个函数，通过key来获取对象的值，通常我们并不知道对象包含的key集合

    1function getObject<T extends object, K extends keyof T>(obj:T, key:K){
2    return obj[key]
3}
4const point = {
5    x: 12,
6    y:13
7}
8getObject(point,'x')
9getObject(point,'z') //类型“"z"”的参数不能赋给类型“"x" | "y"”的参数。
